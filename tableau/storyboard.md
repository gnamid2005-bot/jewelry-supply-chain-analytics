# Tableau Dashboard Storyboard

## Problem This Dashboard Solves

This dashboard helps a viewer understand supply chain performance in an anonymized jewelry manufacturing and retail scenario. It connects monthly fulfillment performance, supplier contribution, product mix, and SKU-level drill-down into one public portfolio story.

The dashboard is designed for questions such as:

- Are delivery quantities keeping up with order quantities?
- Which anonymized factories contribute the most delivery labor value?
- Is delivery value concentrated among a small number of suppliers?
- Which product categories and series drive the largest share of value?
- Which SKU-level patterns deserve a closer look?

## What to Notice First

The viewer should start with the Executive Overview page.

First, look at the KPI cards:

- Total Order Quantity
- Total Delivery Quantity
- Delivery Labor Value
- Fulfillment Rate
- Active SKU Count

Then compare the monthly order and delivery trend. A visible gap between order and delivery is the first signal that the viewer should investigate supplier performance, product mix, or selected SKU behavior.

## How to Move Through the Analysis

1. Start with Executive Overview.
   Use this page to identify high-level trend shifts, fulfillment gaps, and changes in delivery labor value.

2. Move to Supplier Performance.
   Check whether gaps or value changes are associated with specific anonymized factories or factory types. Review top factories and supplier concentration.

3. Move to Product Mix Analysis.
   Understand whether product category, product series, or SKU source is driving changes in delivery labor value or average labor value per unit.

4. Finish with SKU Drill-down.
   Select an anonymized SKU to inspect monthly order, delivery, and delivery labor value trends. Use this page for detailed follow-up after identifying an issue or opportunity in the first three pages.

## Insights the Dashboard Is Designed to Reveal

- Fulfillment pressure by month.
- Delivery value growth or decline.
- Supplier concentration and dependency patterns.
- Differences between in-house, external, partner, unknown, and other factory types.
- Product category and product series contribution to delivery value.
- SKU-level delivery gaps and value patterns.

## Portfolio Framing

This is an independent student portfolio project inspired by internship experience. It uses anonymized and generalized sample data only. The Tableau Public version should not include company names, real supplier names, real SKU identifiers, product images, raw product descriptions, or internal labels.
